## Блок 1 — VKontakte (ВКонтакте)

VK — крупнейшая соцсеть СНГ. Большинство инструментов — веб-сервисы без API; агент рекомендует их пользователю для ручного выполнения.

### Веб-инструменты VK (все — ручные, White)

| Инструмент | Назначение | URL | Ключевая возможность |
|------------|------------|-----|----------------------|
| **220vk.com** | Возраст аккаунта, скрытые друзья, дата регистрации | 220vk.com | Дата создания профиля — критично для верификации |
| **vk5.city4me.com** | Статистика онлайна, скрытые друзья | vk5.city4me.com | Паттерны активности пользователя |
| **vkpt.info** | Лайки, комментарии, старые друзья | vkpt.info | История взаимодействий |
| **TopDB.ru** | Скрытые связи между аккаунтами | topdb.ru | Поиск аффилированных лиц |
| **Looka.one** | Анализ друзей, скрытые связи | looka.one | Граф социальных связей |
| **Kribrum.io** | Поиск удалённых публикаций по ID | kribrum.io | Восстановление удалённого контента |

### Telegram-боты для VK (Grey, не автоматизируемы)

| Бот | Назначение | Команда |
|-----|------------|---------|
| **@VKHistory** | История профиля с 2010, все аватары | Отправить ID или ссылку на профиль VK |
| **@SocialGraph** | Граф связей, группы родственников | Отправить ссылку на профиль |
| **@FindClone** | Поиск дублей по фото в VK | Отправить фото |
| **@ФотоБот** | Поиск аккаунта VK по фото лица | Отправить фото |
| **@FriendlyGraph** | Визуализация графа друзей | Отправить ссылку на профиль |

> ⚠️ **[GREY] Правовой риск: MEDIUM** (РБ, РФ). Достоверность: 60–80%. Кросс-верификация обязательна. Не автоматизируется — только рекомендация пользователю.

### Алгоритм VK-разведки (ручной)

1. Найти профиль в VK (через Sherlock / Snoop Project / ручной поиск)
2. 220vk.com → дата регистрации, ID, скрытые друзья
3. vkpt.info → лайки, комментарии, старые друзья
4. TopDB.ru → скрытые связи с другими аккаунтами
5. Kribrum.io → поиск удалённых публикаций
6. @VKHistory (Telegram) → история аватаров и изменений профиля

## Блок 2 — Face Recognition (поиск по фото)

### Инструменты (все — ручные, кроме отмеченных)

| Инструмент | Что ищет | Специфика | Доступ |
|------------|----------|-----------|--------|
| **search4faces.com** | VK, OK, TikTok, Clubhouse | Лучший для СНГ-соцсетей | Web, бесплатно |
| **vk.watch** | Похожие лица в ВКонтакте | Специализация VK | Web |
| **@FindCloneBot** | Дубли фото в VK | Telegram-бот [GREY] | Отправить фото |
| **@ФотоБотBot** | Аккаунт VK по фото | Telegram-бот [GREY] | Отправить фото |
| **PimEyes** | Похожие лица, клоны в сети | Платный, международный | pimeyes.com |
| **facecheck.id** | Поиск в соцсетях по фото | Ссылки на профили | facecheck.id |
| **Яндекс Картинки** | Поиск по изображению в Яндексе | Глубже по Рунету чем Google Images | yandex.ru/images |

### Алгоритм поиска по фото (ручной)

1. Яндекс Картинки (yandex.ru/images) → поиск по изображению — лучший для СНГ
2. search4faces.com → поиск в VK/OK/TikTok
3. @FindCloneBot → дубли фото в VK
4. PimEyes / facecheck.id → международный поиск

## Блок 3 — YaSeeker (OSINT по Яндекс-аккаунту)

**YaSeeker** — CLI-инструмент для получения информации по учётной записи Яндекса (ник, email). Выдаёт: полное имя, фото, пол, Yandex UID, связанные аккаунты в соцсетях, статистику активности.

| Параметр | Значение |
|----------|----------|
| Запуск | `cd ~/osint/tools/yaseeker && python3 yaseeker.py <username_or_email>` |
| Проверка | `ls ~/osint/tools/yaseeker/yaseeker.py` |
| Метод | White (публичный API Яндекса) |

**Если YaSeeker не установлен:**
- Зафиксируй в «Пробелах»: «YaSeeker не установлен»
- Альтернатива для пользователя: ручной поиск в Яндекс.Картинках + Яндекс.Поиск по нику

**Пример вывода YaSeeker:**
```
Yandex UID: 123456789
Display Name: Иван Иванов
Gender: male
Avatar: https://avatars.mds.yandex.net/...
Social links: vk.com/ivanov, t.me/ivanov
```

## Блок 4 — Поиск по телефону (СНГ)

### tel.belinf.net — White

**tel.belinf.net** — одновременный поиск по номерам BY/RU/UA в нескольких поисковиках, с автоподбором форматов (национальный, международный).

| Параметр | Значение |
|----------|----------|
| URL | tel.belinf.net |
| Тип | Web (ручной, White) |
| Что ищет | Оператор, регион, упоминания в поисковиках, соцсети |
| Регион | Беларусь, Россия, Украина |

**Алгоритм:**
1. Открой tel.belinf.net в браузере
2. Введи номер в любом формате (сервис автоподбирает форматы)
3. Просмотри результаты из разных поисковиков
4. Зафиксируй: оператор, регион, найденные упоминания

### Telegram-боты для телефона (Grey)

## Правовая оговорка

- Веб-инструменты (220vk, search4faces, tel.belinf.net, Яндекс.Картинки) — White: публичные данные, легально.
- YaSeeker — White: публичный API Яндекса, легально.
- Telegram-боты — Grey: правовой риск MEDIUM–HIGH (ФЗ-152 РФ, 99-З РБ). Только после Grey-эскалации.
- Face Recognition: поиск по лицу без согласия субъекта — риск в юрисдикциях с жёстким регулированием ПД (РБ: 99-З; ЕС: GDPR). Для РБ — только с согласия субъекта.
