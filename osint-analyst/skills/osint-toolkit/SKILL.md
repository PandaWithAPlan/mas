> ⚠️ **УДАЛЁН в пользу гранулярных навыков (2026-06-08)**
>
> Этот навык больше не содержит актуального контента. Используй вместо него:

| Для задачи | Загрузи навык |
|---|---|
| Методология White/Grey + эскалация | `osint-methodology` |
| Региональная специфика (СНГ, ЕС) | `osint-regions` |
| Домены, IP, инфраструктура | `osint-tools-infra` (amass, Shodan, crt.sh, dnsrecon, subfinder, assetfinder) |
| Личность, email, username, телефон | `osint-tools-identity` (Sherlock, HIBP, Holehe, phoneinfoga) |
| Grey: darknet, утечки, Telegram-боты, h8mail, trufflehog | `osint-tools-grey` |
| Файлы, метаданные, дорки | `osint-tools-analysis` (ExifTool, VirusTotal, osrframework) |
| Веб-сайты: технологии, URL, скриншоты | `osint-tools-web` (whatweb, gau, waybackurls, eyewitness) |
| Python-шаблоны для автоматизации | `osint-tools-scripts` |
| Модульный OSINT-фреймворк (150+ модулей) | `osint-tools-sn0int` |
| Формирование отчёта (coverage gate, Mermaid) | `osint-reporting` |

Если тебе **абсолютно необходимо** содержимое старого `osint-toolkit` — оно доступно в истории Git.
Но для всех новых задач используй гранулярные навыки: меньше контекста, точнее инструменты, безопаснее изоляция Grey.
